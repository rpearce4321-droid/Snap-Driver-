# Retainer Use-Case SOP

Last updated: 2026-02-08

## Purpose
Guide a Retainer through profile setup, seeker discovery, linking, routes, and work-unit driven scoring.

## Profile setup
1. Complete all required profile fields.
2. Confirm the retainer tier and entitlements.
3. Add any staff users if applicable.

## Seeker discovery
1. Use the Seeker wheel to review approved seekers.
2. Sort by schedule match when needed.
3. Classify seekers into Excellent, Possible, or Not now lists.

## Linking
1. Request links for approved seekers.
2. Complete the video confirmation step.
3. Approve the link when both parties agree.
4. Enable Working Together once the link is active.

## Routes and scheduling
1. Create routes with accurate schedule and timezone.
2. Lock in seekers with a start date.
3. Keep route status updated through the lifecycle.

## Work units and badges
1. Submit completed and missed work units at cadence end.
2. Allow the 48-hour window for neutral or negative confirmation.
3. Use disputes only when required and document notes.

## Posts and broadcasts
1. Create posts and broadcasts within tier limits.
2. Set audience correctly for linked-only or public reach.
